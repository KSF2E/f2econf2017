    十年一覺前端夢

    講者Lock Chou
    十年之前，我不認識「前端」；十年以來，「前端」已經在網站開發領域舉足輕重。 一直以來，我們都熱愛在彈指間舞動程式，玩轉像素，渴望觸及世界彼端的萬千使用者，提供他們更流暢的操作經驗。 這份工作的職稱之多：網頁設計師、軟體工程師、前端（Front-end）還是全端工程師（Fullstack），讓人目不暇給。 這個議程會從「前端」這個名詞的崛起時空談起，兼論技能樹的演化，乃至探討『為什麼我們找不到前端工程師』這篇頗受歡迎的翻譯文章：在目眩神迷的技術與名詞中回看本質，找出適者之道。

    用 RxJS 寫出可讀的程式碼

    講者Jerry Hong
    在網頁需求越來越複雜的今天，非同步行為一直是前端工程師要面臨的挑戰，不只是程式邏輯變得複雜、可讀性變低，還有不同 API 帶來的痛苦。 RxJS 提供了一套完整的非同步解決方案，讓我們在面對各種非同步行為，不管是 Event, AJAX, 還是 Animation 等，我們都可以使用相同的 API 做開發，並且同時保有很高的可讀性。
